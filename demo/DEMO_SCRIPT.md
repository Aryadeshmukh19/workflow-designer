# 2-minute demo script (what to click and say)

1. Open the app (http://localhost:5173) — show the canvas and sidebar.
2. Drag **Start** → **Task** → **Automated** → **End** into the canvas.
3. Connect Start -> Task -> Automated -> End using the connector dots.
4. Click **Automated** node, choose action **Send Email**, fill `to` with `test@example.com` and Save.
5. Click **Run Simulation** — explain: "The simulator traverses nodes, detects cycles, and returns a step-by-step log".
6. Click **Export JSON** and mention you can validate with the Rust validator or run FastAPI alternative.
7. Show `APPLICATION_REPLY.txt` (submission text) and README badge highlighting bonus tech.

Speak clearly: mention technologies used (React+TS, Tailwind, Zustand, Express/FastAPI, Rust) and that the repo includes example workflows and docker-compose for easy demo.
